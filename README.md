# GitHub-Portal
Github-Portal is an github user searching app which comes up with users searching and finding their respective Profile through GitHub API.

#### Deployment Website Link - [https://github-portal.netlify.app](https://github-portal.netlify.app)

## Features
Github-Portal have features as follows - 
* You can search through username. Also we have Advanced option where we can get the desired set of users
* You can look for every Users GitHub profile and also their respective public Repositories.
* You can look for users Respective GitHub Profile and their respective public Repositories in Github by Links.
* You can change the display mode of App i.e, Light/Dark mode as per your need.
* Fully Responsive App with support from Desktop to Mobile Devices.

## Frameworks used -
### Frontend Frameworks- 
#### HTML,CSS,React,Redux,Formik,React-Router,Materiul-UI

## How to Run the Application
1. Either Fork or download the app and open the folder in the CLI.
2. Install all the dependencies using `npm i` command.
3. Start the Application using command `npm start` .The app will be served at http://localhost:3000
4. Go to http://localhost:3000
